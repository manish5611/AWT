Program Name
Program 2 - JavaScript Registration Form Validation

The lab question.
Write JavaScript to validate the fields of a Registration page (First Name, Last Name, Email, Password, Mobile Number, Address).

Folder Structure
Program2
│── index.html

Steps to Run
1. Open index.html in a web browser.
2. Fill the form and submit to see validations.

Expected Output
A form that displays JavaScript alert popups if validation rules are broken, and a success alert if all inputs are valid.

Common Errors
Validation not triggering: Check for typo in the onsubmit="return validateForm()" attribute.
