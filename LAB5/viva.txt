Viva Questions for Program 5:

1. How do you handle form inputs in React?
Answer: Using Controlled Components. You bind the input's 'value' attribute to a state variable, and use the 'onChange' event handler to update the state using e.target.value.

2. What does Object.keys(error).length === 0 check?
Answer: It checks if the error object has no properties, which means no validation errors were found.

3. Why use .trim() on string inputs?
Answer: It removes whitespace from both ends of a string, preventing users from bypassing validation by just entering spaces.

4. Explain controlled vs uncontrolled components.
Answer: A controlled component has its state managed by React (via useState). An uncontrolled component manages its own state internally, and you typically access its value using a Ref.

5. How do you show conditional error messages?
Answer: By checking if the error string exists and rendering it if it does, often using conditional rendering or directly injecting the state variable like {errors.firstName} in JSX.
