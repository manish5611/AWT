Program Name
Program 1 - HTML Tags, Links, Images, Tables, Lists, Frames

The lab question.
Create a department page using the following HTML tags. a) Text Formatting tags b) Links c) Images d) Tables e) list f) frames

Folder Structure
Program1
│── header.html
│── footer.html
│── menu.html
│── home.html
│── image.html
│── list.html
│── table.html
│── main.html

Steps to Run
1. Open the folder containing the HTML files.
2. Double click on main.html to open it in a web browser.

Expected Output
A webpage split into three frames (header, footer, and a middle section with a menu on the left and content on the right).

Common Errors
Image not showing: Make sure "department.jpg" is in the same folder.
