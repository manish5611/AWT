Program Name
Program 3 - Class Components, Events, and Lifecycle Methods

The lab question.
a) Develop a reactJs Application which demonstrate the working of Class component to display Employee details like Sname , Sid, Dept, Age, Apply inline CSS styling for the same
b) Develop a program to handle user events
c) Develop a React application that includes multiple components. Understand and implement the React component lifecycle methods.

Folder Structure
Program3
│── src
│   │── App.js
│   │── Employee.jsx
│   │── Event.jsx
│   │── Lifecycle.jsx

Steps to Run
1. Setup a React app with Vite or Create React App.
2. Copy Employee.jsx, Event.jsx, and Lifecycle.jsx to your src folder.
3. Edit your App.jsx to import and render these components.
4. Run npm install and npm start (or npm run dev).

Expected Output
Components rendering Employee Details, Event handling with alerts and inputs, and Lifecycle methods printing in the browser console.

Common Errors
Image not found: In Event.jsx, "/mother.JPG" might not exist in your public folder. Place an image there or change the source.
